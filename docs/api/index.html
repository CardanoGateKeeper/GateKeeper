Authenticating requests

This API is not authenticated.

Public API

List public, discoverable events.

Returns upcoming public events along with their associated Cardano policy IDs. Clients can use this to find events that a connected wallet might qualify for.

Example request:
curl --request GET \
    --get "http://localhost/api/events/discover?policy_hashes%5B%5D=" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/events/discover"
);

const params = {
    "policy_hashes[]": "",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Request   

GET api/events/discover

Headers

Content-Type        

Example: application/json

Accept        

Example: application/json

Query Parameters

policy_hashes[]   string[]  optional    

Optional. One or more Cardano policy IDs to filter by. Only events that accept at least one of these policies will be returned.

Response

Response Fields

data   object     
uuid   string     

The event UUID used in app links.

name   string     

The display name of the event.

description   string|null     

A short description of the event.

location   string|null     

The event venue or location.

date   string|null     

Event date in Y-m-d format.

start   string|null     

ISO8601 start datetime.

end   string|null     

ISO8601 end datetime.

policies   object     
hash   string     

Cardano policy ID accepted for this event.

name   string     

Human-readable label for the policy.

policy_hashes   string[]     

Convenience array of all policy hashes for this event.

Discover public events for a specific policy.

Returns upcoming public events that accept the given Cardano policy ID. This is a convenience wrapper around the main discover endpoint with a single policy filter.

Example request:
curl --request GET \
    --get "http://localhost/api/events/discover/policy/a0028f35d4c7b4f2f0b1d6a0e4c3a9f6d0b0cafe" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/events/discover/policy/a0028f35d4c7b4f2f0b1d6a0e4c3a9f6d0b0cafe"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
